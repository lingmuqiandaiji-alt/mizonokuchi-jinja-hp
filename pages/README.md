# pages/ フォルダについて

第2回以降で再構築する下層ページの作業用フォルダです。

## 重要：公開時の配置ルール

既存URL保全のため、**完成した下層ページはサイトのルート（index.htmlと同じ階層）に、既存と同じファイル名で配置します**（例：gosaijin.html, gokitou.html）。
`pages/` はあくまで作業・下書き用であり、公開URLを `pages/〜` に変更してはいけません。
